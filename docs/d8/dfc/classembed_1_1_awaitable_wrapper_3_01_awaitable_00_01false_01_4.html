Detailed Description

template<class Awaitable>
requires (!std::derived_from<Awaitable, AwaitableNode>)
class embed::AwaitableWrapper< Awaitable, false >

Wraps awaitables that are not yet derived from embed::AwaitableNode

If wraps arbitrary awaitables that do not derive from embed::AwaitableNode that implements the registration infrastructure relied on by embed::Coroutine and embed::Task.

Specialises for awaitable passed as rvalues.

Intended to be used only by CoroutinePromise::await_transform(awaitable)

Template Parameters
Awaitablean awaitable that implements .await_ready(), .await_resume() and .await_suspend(handle).
