Detailed Description

A real time task is time aware. It has an assigned clock, a ready time and a deadline.

Real-time tasks can be used in real-time schedulers The real-time schedulers derives from Task and additionally provides a ready-time. The RealTimeScheduler will only start the task after the ready time. Further, it provides a deadline. A real-time scheduler will prioritise task with sooner deadlines.

In case you like to create your own real-time task and derive from this one, it provides the following customisations:

  • Overload the following to calculate the next schedule from the last schedule and/or execution times.
